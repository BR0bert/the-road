1. Enter name of the new to-do card in the input form and click ADD CARD

2. The new card will show up it's title/name (h2.title) being the one inserted.

3. To add tasks to the card enter the new task in the input form and press ENTER.
The new task will show up in the card's body.

4.To delete the card simply click "Delete Card" bottom right corner.

5.To remove all the completed tasks click "Clear completed tasks" bottom left corner.
